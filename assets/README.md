# Frog Crossing Sprites

Put your sprite PNGs in this folder to override the built-in generated sprites.

Expected filenames:

- `frog.png`
- `croc.png`
- `fly.png`
- `log.png`
- `lilypad.png`

Notes:

- Use `.png` with transparency if possible.
- The game will automatically scale images to fit.
- If a file is missing, the game uses a generated placeholder sprite.
